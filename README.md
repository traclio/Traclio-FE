# Traclio-FE

## Tech Stack

- React
- TypeScript

## Commit Convention

| Type | Description |
|:--------|:-------------|
| feact | 새 기능 추가 |
| fix | 버그 수정 |
| refactor | 코드 리팩토링 |
| docs | 문서 추가 또는 수정 |
| test | 테스트 코드 작성 | 
| delete | 불필요한 코드나 파일 삭제 |
| build | 빌드 관련 파일 수정|
| chore | 그 외 자잘한 수정 |

## Example

```bash
feat: 로그인 기능 추가
fix: 회원가입 오류 수정
